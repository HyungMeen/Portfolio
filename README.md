### 📌 MCU별 프로젝트 분류

#### 🔷 MEC1521

| 프로젝트명 | 설명 | 링크 |
|------------|------|------|
| UART Logger | MEC1521의 UART를 사용하여 외부 장치 디버깅 및 로그 수집
| GPIO Toggle | MEC1521의 GPIO를 주기적으로 토글하여 LED 점멸 확인

---

#### 🔷 ATmega128A

| 프로젝트명 | 설명 | 링크 |

| UART 7-Segment Display | UART 수신값을 7세그먼트에 표시
